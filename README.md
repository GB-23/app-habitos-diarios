# Projeto AV2 Mobile!


- Esse projeto foi feito para o trabalho avaliativo (AV2) de Mobile 2025.2!

- Para navegação entre telas foi usado o Expo Router

- Para armazenamento local foi usado o Async Storage

- Para gerenciamento de tema (claro/escuro) foi usado Context API do React

- A estrutura do projeto é simples, com componentes funcionais e hooks do React

- O foco principal foi em funcionalidades básicas de hábitos diários, como adicionar, editar, deletar e marcar como concluído

- O design é simples e funcional, com temas claro e escuro
